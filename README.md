# 🟩 Wordle CLI (Go)

A simple Wordle game built in Go that runs in the terminal.

## 🚀 Features

- 5-letter word guessing game
- Limited number of attempts
- Feedback system (correct letters & positions)
- Tracks user stats (games played, wins, etc.)
- Reads words from a file

## 🛠 Tech Stack

- Go (Golang)
- CLI (Terminal-based interface)

## 📦 Project Structure

.
├── main.go        # Entry point  
├── game.go        # Game logic  
├── user.go        # User and stats logic  
├── input.go       # Input handling  
├── wordle-words.txt # Word list
